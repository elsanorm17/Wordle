# Wordle
This project is a recreation of the New York Times game Wordle using Python. It includes the basic functionalities of the game but is meant to be replayable, with a randomly selected answer each time the code is run.

Players have up to 6 chances to guess a 5-letter word, receiving feedback for each guess in the form of colored tiles:
